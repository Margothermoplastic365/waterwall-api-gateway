API Gateway Architecture

1. Platform Overview 2. Identity Service 3. API Protection (4 Auth Types) 4. Two-Tier Cache Strategy

Platform Overview Architecture

Tech Stack: Spring Boot 3.x + Java 21 (MVC) | React + Next.js | PostgreSQL + pgvector + Caffeine | RabbitMQ | Docker Compose | No Redis

Identity Service — Internal Architecture

Spring Boot microservice: Spring Authorization Server + custom modules. Sessions in PostgreSQL, cache in Caffeine. No Redis.

API Protection — 4 Authentication Type Flows

API Key, OAuth2/JWT, mTLS, Basic Auth. Per-API auth policy. Caffeine cache (no Redis).

Two-Tier Cache Architecture

Tier 1: Caffeine local cache + RabbitMQ invalidation (99% of ops). Tier 2: PostgreSQL revocation list + atomic counters (security-critical). No Redis.
